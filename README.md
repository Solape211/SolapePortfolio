## Development

```bash
# install Gridsome globally
yarn global add @gridsome/cli

# clone repository
git clone https://github.com/jeferson-sb/portfolio.git

# cd into the project directory
cd portfolio

# install dependencies
yarn

# Start local dev server
gridsome develop
```
